## Installation

Clone the repository to your local machine.

```
https://github.com/Hasib192/react-accordion.git
```

Navigate to the project directory.

```
cd react-accordion
```

Install the required dependencies.

```
npm install
```

Start the development server:

```
npm run dev
```
